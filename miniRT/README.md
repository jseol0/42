## How to
```
make && ./miniRT *.rt
make all && ./miniRT *.rt
```

- Camera Rotation : `Space Bar`
- Camera Zoom : `Mouse Wheel`
- Camera Tilt : `Arrow Key`
- Close : `ESC`
